# 📄 Bangladesh NID Server Copy Generator

A dynamic, modern, and client-side web application to generate a printable Bangladesh National ID (NID) server copy report instantly. This project features a stylized, colorful form that processes data locally and renders it into an optimized A4 print layout.

🌍 **Live Demo:** [https://mojiibrsm.github.io/NID-Server-copy/](https://mojiibrsm.github.io/NID-Server-copy/)

---

## ⚡ Features

- **Dynamic Form System:** Sleek and colorful user interface built with linear gradients and modern web typography.
- **Direct Photo Upload:** No image URL required! Users can upload images directly from their device via a local file reader.
- **Mobile Number Integration:** Included a dedicated input field for tracking mobile data.
- **Auto-Updated QR Code:** Generates a real-time QR code combining Name, NID, DOB, and Mobile number.
- **Print-Ready Layout:** Built-in A4 CSS optimization that automatically hides forms/buttons during printing.
- **Security Feature:** Disabled right-click context menu on the preview page.

---

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3 (Flexbox & Grid Layout)
- **Scripting:** Pure JavaScript (ES6+) & jQuery
- **Fonts & Icons:** Google Fonts (Tiro Bangla & Poppins), FontAwesome v6
- **API Used:** QRServer API for real-time QR code generation
